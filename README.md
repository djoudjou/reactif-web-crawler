# reactif-web-crawler

Projet visant à implémenter un web-crawler d'après la phylosophie Reactif-programming + ATDD


## Build
mvn clean package

## Run
java -jar ./target/reactif-web-crawler-{version}.jar

## Usage
curl 'http://localhost:9000/crawl?depth=4&url=http://foat.me/
